# 00 — Objetivo

## O que se quer alcançar

Plataforma web de suporte à cooperação institucional entre Recife e Nantes.
Permite consolidar informações dispersas (internet, Google Drive, arquivos
locais), mapear os atores envolvidos dos dois lados, visualizar conexões entre
eles e facilitar a comunicação e troca de documentos entre as delegações.

- **Setorial:** industria criativa, tecnologia/digital, saude

- **Publico geral (read-only):** qualquer pessoa com o link

## Escopo dentro

- Mapa visual de atores e conexoes (MVP)
- Diretorio de atores/contatos com perfis
- Upload de documentos (agendas, apresentacoes, arquivos) pelos membros
- Busca/pesquisa sobre a cooperacao
- Agenda de eventos/missoes
- Area de comunicacao entre as partes
- Consolidacao de documentos
- Reprocessamento periodico da base (manual, via Claude Code)

## Escopo fora

- APIs de IA no runtime (processamento eh offline)
- Dominio proprio (usa .vercel.app)
- Servicos pagos
- App mobile nativo (web responsivo basta)
- Chat em tempo real (comunicacao eh assincrona)

## Criterio de sucesso

1. **Intermediario:** membros das delegacoes conseguem consultar o mapa e
   encontrar quem eh quem dos dois lados
2. **Final:** 10 pessoas usando ativamente a plataforma
